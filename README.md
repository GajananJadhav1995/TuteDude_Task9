# Laundry Services Hero Section – Hover Effect Assignment

##  Project Overview

This project is a simple responsive Laundry Services landing page built using HTML and CSS.
The main purpose of this assignment is to implement a hover effect using CSS on the “Book Your Service Now” button.

The overall website structure such as Navbar, Hero Section, and layout was created in a previous assignment.
In this assignment, the focus is only on adding a hover animation to the button using the `:hover` pseudo-class and `transform` property.

##  Hover Effect Implementation

The hover effect is applied to the Book Service button using CSS.

When the user hovers over the button:

 The button rotates slightly
 The width increases
 A smooth transition effect is applied

### CSS Code

```css
.book-service:hover{
    transform: rotate(-5deg);
    font-size: 18px;
    width: 35%;
    transition: 0.6s ease;
}
```
